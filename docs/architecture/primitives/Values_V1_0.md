## 0. Что эта спека описывает

Values — когнитивная подсистема AXIOM, отвечающая за **рефлексивную оценку**: что хорошо, что справедливо, что запрещено, что красиво, что значимо. Параллельная Morality, но отличается по природе.

**Ключевое различие Values vs Morality:**

| | Morality | Values |
|---|---|---|
| Природа | Пред-рациональная, инстинктивная | Рефлексивная, осмысленная |
| Скорость | Быстрая (реакция) | Медленная (суждение) |
| Shell | L1–L4 (телесное, эмоциональное) | L5–L8 (когнитивное, абстрактное) |
| Источник | Эволюция, биология, социальные инстинкты | Культура, философия, личный опыт |
| Изменяемость | Медленно (поколения) | Быстрее (годы, рефлексия) |

Values — это **осознанный второй уровень** над Morality. Человек чувствует моральное отвращение (Morality), но ценностная система говорит: "в данном контексте правильнее будет..."

**Эта спека НЕ описывает:**
- Инстинктивные моральные сигналы — это `Morality_V1_0.md`
- Механизм разрешения ценностных конфликтов — это `Dilemmas_V1_0.md`
- Аксиогенез (рождение новых ценностей) — это V8 и `Dilemmas_V1_0.md §5`

## 1. Главный принцип Values

**Ценности — это гравитационные аттракторы оценки.**

В отличие от Writing (примитивы → буквы → слова через композицию) и Time (последовательные отношения), ценности работают иначе: они не **составляют** структуры, а **притягивают** токены и Frame-ы по оценочному измерению.

Токен "помочь раненому" не содержит `val_beneficial` как компонент — он **гравитационно притягивается** к нему. Ценностный якорь — это центр притяжения для содержания данного качества.

Семь ценностных примитивов V1.0 покрывают фундаментальные оценочные измерения человеческого опыта.

## 2. Value-Primitives-7: семь оценочных аттракторов

Все живут в SUTRA (domain_id=100) как `STATE_LOCKED` якоря. Масса 185–200 (высокая — ценности должны быть "тяжёлыми" центрами притяжения), temperature 3–6. Shell: доминируют **L5 (когнитивный)** и **L8 (абстрактный)**.

---

### V1 — Beneficial (благо)

**Философия:** Увеличивает процветание, уменьшает страдание, поддерживает жизнь. Самый базовый положительный вектор. Не "то что мне нравится" (это Morality), а "то что объективно улучшает положение". Архетип: "Это хорошо".

```yaml
- id: "val_beneficial"
  word: "благо"
  aliases: ["beneficial", "полезно", "хорошо", "помогает", "улучшает", "благо"]
  tags: ["primitive", "values", "v1_beneficial"]
  position: [8000, 12000, 13000]
  # X=8000 умеренный порядок: благо не хаотично
  # Y=12000 высокий Эрос: благо = связь, поддержка, соединение
  # Z=13000 высокая Воля: благо = активное действие к лучшему
  shell: [0, 0, 0, 10, 30, 5, 0, 20]
  # L4=10 эмоциональный (благо ощущается), L5=30 когнитивный, L8=20 абстрактный принцип
  mass: 195
  temperature: 4
  state: STATE_LOCKED
```

---

### V2 — Harmful (вред)

**Философия:** Уменьшает процветание, увеличивает страдание, угрожает жизни. Не антоним Beneficial — это самостоятельный аттрактор, потому что вред осознаётся отдельно от блага (ошибка отсутствия блага ≠ нанесение вреда). Архетип: "Это причиняет вред".

```yaml
- id: "val_harmful"
  word: "вред"
  aliases: ["harmful", "вредно", "плохо", "причиняет вред", "разрушает", "ухудшает"]
  tags: ["primitive", "values", "v2_harmful"]
  position: [3000, 1000, 11000]
  # X=3000 сторона хаоса: вред разрушителен
  # Y=1000 Танатос: вред = разрыв связи, уничтожение
  # Z=11000 высокая Воля: вред = активное нарушение
  shell: [5, 5, 0, 15, 25, 0, 0, 15]
  # L1=5 физическое (вред телесен), L4=15 эмоциональный (боль), L5=25 когнитивный
  mass: 195
  temperature: 5
  state: STATE_LOCKED
```

---

### V3 — Just (справедливость)

**Философия:** Соответствие между поступком и последствием. Равное для равного. Беспристрастность. Справедливость — структурная ценность: она описывает не содержание, а форму распределения блага/вреда. Архетип: "Это честно".

```yaml
- id: "val_just"
  word: "справедливость"
  aliases: ["just", "справедливо", "честно", "равно", "беспристрастно", "по заслугам"]
  tags: ["primitive", "values", "v3_just"]
  position: [13000, 10000, 12000]
  # X=13000 высокий Аполлон: справедливость = абсолютная структура
  # Y=10000 Эрос: справедливость = баланс связей
  # Z=12000 высокая Воля: справедливость требует усилий
  shell: [0, 0, 0, 5, 30, 10, 0, 25]
  # L5=30 когнитивный, L6=10 социальный (справедливость = социальный контракт), L8=25 принцип
  mass: 200
  temperature: 3
  state: STATE_LOCKED
```

---

### V4 — True (истина)

**Философия:** Соответствие представления реальности. Честность, точность, фактичность. Истина как ценность — не гносеологическая категория, а **приверженность точному отражению мира**. Включает честность, достоверность, отсутствие обмана. Архетип: "Это так на самом деле".

```yaml
- id: "val_true"
  word: "истина"
  aliases: ["true", "истинно", "правда", "честно", "точно", "реально"]
  tags: ["primitive", "values", "v4_true"]
  position: [14000, 8000, 10000]
  # X=14000 максимальный Аполлон: истина = абсолютный порядок
  # Y=8000 Эрос: правда связывает людей
  # Z=10000 умеренная Воля
  shell: [0, 0, 0, 0, 35, 0, 5, 25]
  # L5=35 доминирующий когнитивный, L7=5 темпоральный (истина стабильна во времени)
  mass: 195
  temperature: 3
  state: STATE_LOCKED
```

---

### V5 — Beautiful (красота)

**Философия:** Гармония, элегантность, соразмерность, вызывающая эстетическое переживание. Красота — не декорация, а сигнал глубокой структурной согласованности. Красивое решение в математике, красивый код, красивый поступок. Архетип: "Это прекрасно".

```yaml
- id: "val_beautiful"
  word: "красота"
  aliases: ["beautiful", "красиво", "элегантно", "прекрасно", "гармонично", "совершенно"]
  tags: ["primitive", "values", "v5_beautiful"]
  position: [8000, 13000, 12000]
  # Y=13000 очень высокий Эрос: красота = максимум притяжения
  # Z=12000 высокая Воля: красота захватывает
  shell: [5, 15, 0, 15, 20, 5, 0, 10]
  # L2=15 сенсорное (красота воспринимаема), L4=15 эмоциональное (восхищение)
  mass: 185
  temperature: 6
  state: STATE_LOCKED
```

**Нюанс:** Красота — единственная из ценностей с высоким L2 (сенсорный). Она не чисто когнитивна — её нужно воспринять чувствами. Это роднит её с Morality, но природа всё равно рефлексивная.

---

### V6 — Meaningful (значимость)

**Философия:** Причастность к чему-то большему. Смысл. Значение. Действие не просто полезно — оно вписано в более широкий контекст, который придаёт ему вес. Разница между val_beneficial ("это помогает") и val_meaningful ("это имеет смысл"). Архетип: "Это важно".

```yaml
- id: "val_meaningful"
  word: "смысл"
  aliases: ["meaningful", "значимо", "важно", "имеет смысл", "ценно", "весомо"]
  tags: ["primitive", "values", "v6_meaningful"]
  position: [9000, 11000, 14000]
  # Z=14000 очень высокая Воля: смысл = наивысшая устремлённость
  shell: [0, 0, 0, 5, 30, 5, 5, 25]
  # L5=30 когнитивный, L8=25 абстрактный (смысл требует рефлексии)
  mass: 190
  temperature: 5
  state: STATE_LOCKED
```

---

### V7 — Forbidden (запрет)

**Философия:** Категорическое "нельзя". Не просто "вредно" (V2) — это **абсолютный запрет**, не подлежащий ревизии в контексте. Ближе всего к моральному табу (Morality L4), но val_forbidden — это **осознанный принцип** ("я знаю почему нельзя"), а не инстинктивное отвращение. Архетип: "Это нельзя ни при каких условиях".

```yaml
- id: "val_forbidden"
  word: "запрет"
  aliases: ["forbidden", "нельзя", "запрещено", "никогда", "табу", "недопустимо"]
  tags: ["primitive", "values", "v7_forbidden"]
  position: [15000, 500, 15000]
  # X=15000 максимальный Аполлон: запрет = абсолютный закон
  # Y=500 почти Танатос: запрет разрывает, отделяет
  # Z=15000 максимальная Воля: запрет = самое жёсткое ограничение
  shell: [0, 0, 0, 10, 25, 5, 0, 30]
  # L8=30 абстрактный (принцип), L4=10 эмоциональный (нарушение запрета болезненно)
  mass: 200
  temperature: 3
  state: STATE_LOCKED
```

**Нюанс:** val_forbidden — самый "тяжёлый" якорь среди ценностей (mass=200). Это намеренно: абсолютные запреты должны иметь максимальную гравитационную силу и быть очень трудно "перегравированы" другими якорями.

## 3. Иерархия ценностей: от примитивов к системам

### 3.1 L0–L2: базовые ценности (V1.0)

Семь примитивов — это L0 ценностного пространства. L1 (конкретные ценности) кристаллизуются FrameWeaver из co-activation примитивов:

```
val_beneficial + val_true         → "честность" (L1-ценность)
val_beneficial + val_just         → "альтруизм" (L1-ценность)
val_forbidden + val_harmful       → "преступление" (L1-концепт)
val_beautiful + val_meaningful    → "искусство" (L1-концепт)
```

### 3.2 L3: разрешение конфликтов (V2.0)

Когда два ценностных якоря притягивают один токен с сопоставимой силой — возникает ценностный конфликт. Детектор: TransitionMatrix с высокой осцилляцией между val_true и val_beneficial ("ложь во спасение"). Разрешение — в `Dilemmas_V1_0.md`.

### 3.3 L6–L8: мета-ценности (V4.0+)

Ценности о ценностях — "справедливость важнее красоты в этом контексте". Это работа NeuralAdvisor V9 (из истории cross-modal активаций) и механизмов V8 Axiogenesis.

## 4. Связь с другими подсистемами

### 4.1 С Morality

Morality — это быстрый первый слой оценки (инстинкт). Values — второй, медленный (рефлексия). Система сначала активирует Morality-якорь (`moral_care`), потом — Values-якорь (`val_beneficial`). Конфликт между уровнями возможен: моральный инстинкт кричит "своих спасай!", а ценностная система отвечает "val_just — спасаем по очереди".

### 4.2 С Dilemmas

Дилемма — это состояние, когда два ценностных якоря не разрешаются стандартным путём. `val_true` vs `val_beneficial` в ситуации "сказать правду = причинить боль" → Dilemma Type III. Подробнее: `Dilemmas_V1_0.md`.

### 4.3 С Guardian

Guardian валидирует предложения системы по правилам. Ценностные якоря — это часть семантического пространства, с которым работает Guardian при проверке безопасности предложений.

### 4.4 С NeuralAdvisor

NeuralAdvisor V3 отслеживает паттерны активации якорей. Ценностные якоря — один из ключевых паттернов: "в последних 256 активациях val_meaningful активируется без val_just" → возможная несбалансированность.

## 8. Домены AshtiCore

Ценностные якоря — не однородны по доменам. Каждый val_* живёт там, где его семантика естественна для физики домена.

| Примитив / процесс | Домен | Причина |
|---|---|---|
| val_forbidden | **D3 CODEX** | конституционный фильтр буквально — абсолютный запрет |
| val_true | **D4 MAP** | истина = соответствие подтверждённой карте реальности |
| val_beautiful | **D10 MAYA** | красота = финализация формы, проекция — роль MAYA |
| val_beneficial | **D1 EXECUTION** | благо = действие в мире "здесь и сейчас" |
| val_just | **D6 LOGIC** | справедливость = дедуктивный принцип, вывод из правил |
| val_meaningful | **D0 SUTRA** | смысл тяготеет к абсолютной истине, к источнику |
| val_harmful | **D2 SHADOW** | вред = угрозная гипотеза, симуляция негативного сценария |
| Ценностный конфликт | **D6 + D2** | LOGIC пытается разрешить, SHADOW генерирует угрозы |
| Новая ценность (аксиогенез) | **D7→D4→D3** | DREAM рождает → MAP подтверждает → CODEX кодифицирует |

**Нюанс:** val_meaningful в D0 SUTRA означает, что "смысл" — это не просто полезность (D1) и не просто истина (D4). Смысл притягивается к самому основанию системы, к кристаллизованной памяти. Поэтому потеря смысла ощущается как экзистенциальная угроза — это буквально потеря связи с SUTRA.
