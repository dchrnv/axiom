## 0. Что эта спека описывает

Time — когнитивная подсистема AXIOM, отвечающая за восприятие и понимание временных структур. Параллельная Writing и Mathematics, но описывает не форму и не структуру отношений, а **темпоральный характер событий**: последовательность, длительность, причинность, цикличность, горизонт планирования.

Shell V3.0 уже содержит Layer L7 (темпоральный). Time-примитивы — это якоря, которые **наполняют L7 конкретным содержанием**: они дают SUTRA «язык времени», на котором FrameWeaver, ContextRecognizer и ActivityTrace строят временные структуры.

**Эта спека НЕ описывает:**
- COM (Causal Order Model) — системная причинность тиков; см. `CAUSAL ORDER MODEL (COM).md`
- CausalFrontier — причинность на уровне связей; см. `Causal_Frontier_V2_0.md`
- Heartbeat — тактовый генератор; см. `Heartbeat_V2_0.md`
- ActivityTrace — структура коротких/средних/длинных временных окон (это реализация, не примитивы)

Все перечисленное — системная механика времени. Time описывает **когнитивную семантику времени** — как AXIOM "думает" о времени в содержании.

## 1. Главный принцип Time

**Время — это структура отношений между событиями.**

Не абсолютная шкала, не тактовый генератор, а набор когнитивных примитивов, через которые система различает: что было раньше, что позже, что повторяется, что необратимо, куда направлено.

Семь темпоральных примитивов V1.0 охватывают уровни L0–L4 понимания времени (последовательность → планирование). Уровни L5–L8 (релятивистское, ветвящееся, петли) — задел V3–V5.

## 2. Time-Primitives-7: семь темпоральных атомов

Все живут в SUTRA (domain_id=100) как `STATE_LOCKED` якоря. Масса 170–200, temperature 3–7, позиции в `[0..32767]³`. Shell: доминирует **L7 (темпоральный)**.

---

### T1 — Before (до)

**Философия:** Асимметричное отношение предшествования. A происходит — B ещё нет. Порядок не симметричен: то что "до" уже случилось и неизменно. Архетип: "Сначала было".

```yaml
- id: "time_before"
  word: "до"
  aliases: ["before", "прежде", "предшествует", "раньше", "сперва", "сначала"]
  tags: ["primitive", "time", "t1_before"]
  position: [6000, 2000, 8000]
  # X=6000 умеренный Аполлон: последовательность структурна, но не абсолютна
  # Y=2000 слабый Танатос: предшествование — не близость, а дистанция во времени
  # Z=8000 умеренная Воля: прошлое оказывает влияние, но не управляет
  shell: [0, 0, 0, 0, 15, 0, 40, 10]
  # L5=15 когнитивный (осознание порядка), L7=40 темпоральный, L8=10 абстрактный
  mass: 190
  temperature: 4
  state: STATE_LOCKED
```

---

### T2 — After (после)

**Философия:** Зеркало Before. Событие A происходит когда B уже существует в прошлом. В отличие от Before — направлено в будущее, несёт возможность. Архетип: "Потом настало".

```yaml
- id: "time_after"
  word: "после"
  aliases: ["after", "затем", "следует", "позже", "потом", "вслед за"]
  tags: ["primitive", "time", "t2_after"]
  position: [6000, 2000, 10000]
  # X, Y аналогичны T1; Z=10000 выше — будущее несёт больше воли/потенциала
  shell: [0, 0, 0, 0, 15, 0, 40, 10]
  mass: 190
  temperature: 4
  state: STATE_LOCKED
```

**Нюанс:** T1 и T2 — один примитив или два? Семантически это одно отношение ("порядок"), но они разного знака. Два отдельных якоря дают FrameWeaver возможность различать направление, а TransitionMatrix — отслеживать паттерны before→after отдельно от after→before.

---

### T3 — Simultaneous (одновременно)

**Философия:** Два события без различимого порядка. Не причинность, не последовательность — соприсутствие. Важный примитив для мультимодального связывания: TextPerceptor и VisionPerceptor активируются "в один тик" → T3 свидетель этой встречи. Архетип: "В этот момент".

```yaml
- id: "time_simultaneous"
  word: "одновременно"
  aliases: ["simultaneous", "одновременный", "в то же время", "concurrently", "вместе"]
  tags: ["primitive", "time", "t3_simultaneous"]
  position: [8000, 10000, 9000]
  # Y=10000 высокий Эрос: одновременность = связь двух вещей в моменте
  # X=8000 умеренный порядок: сопряжённость требует синхронизации
  shell: [0, 0, 0, 0, 10, 5, 40, 5]
  # L6=5 социальный (совместное присутствие — это уже встреча)
  mass: 180
  temperature: 5
  state: STATE_LOCKED
```

---

### T4 — Duration (длительность)

**Философия:** Не момент, а отрезок. Событие занимает время — у него есть начало и конец, и между ними — протяжённость. Duration — это "тело" времени, в отличие от T1/T2 (скелет) и T3 (точка встречи). Архетип: "Это длится".

```yaml
- id: "time_duration"
  word: "длительность"
  aliases: ["duration", "длина", "протяжённость", "interval", "span", "пока"]
  tags: ["primitive", "time", "t4_duration"]
  position: [5000, 5000, 7000]
  # Нейтральные X и Y: длительность сама по себе не ценностно окрашена
  shell: [0, 5, 0, 0, 15, 0, 45, 10]
  # L7=45 доминирует (длительность — чистое время), L2=5 сенсорное (чувство течения)
  mass: 185
  temperature: 5
  state: STATE_LOCKED
```

---

### T5 — Periodic (цикличность)

**Философия:** Паттерн, возвращающийся через интервал. Суточный цикл, дыхание, сезоны, ритм. Цикличность — это предсказуемость во времени, и одновременно — отличие живого от мёртвого (жизнь пульсирует). Архетип: "Снова приходит".

```yaml
- id: "time_periodic"
  word: "цикл"
  aliases: ["periodic", "цикличность", "ритм", "rhythm", "cycle", "повтор", "снова"]
  tags: ["primitive", "time", "t5_periodic"]
  position: [10000, 7000, 9000]
  # X=10000 высокий Аполлон: цикл структурен, повторяем, законен
  # Y=7000 Эрос: возврат = связь с прошлым собой
  shell: [0, 0, 0, 0, 20, 0, 35, 5]
  # L5=20 когнитивный (распознавание паттерна = когнитивный акт)
  mass: 185
  temperature: 6
  state: STATE_LOCKED
```

**Связь с TransitionMatrix:** decay-статистика переходов между активностями — это прямая реализация T5 на уровне ContextRecognizer. TransitionMatrix накапливает "периодические" паттерны.

---

### T6 — Irreversible (необратимость)

**Философия:** Стрела времени. Яйцо не склеишь. Прошлое неизменно. Энтропия. Это самый фундаментальный из семи — он задаёт асимметрию T1/T2 (почему "до" и "после" не симметричны). Архетип: "Нельзя вернуть".

```yaml
- id: "time_irreversible"
  word: "необратимо"
  aliases: ["irreversible", "необратимость", "entropy", "нельзя вернуть", "навсегда"]
  tags: ["primitive", "time", "t6_irreversible"]
  position: [12000, 1000, 14000]
  # X=12000 высокий Аполлон: необратимость — закон, абсолютная структура
  # Y=1000 слабый Танатос: что ушло — ушло, разрыв связи
  # Z=14000 очень высокая Воля: необратимость = сильнейшее ограничение
  shell: [5, 0, 0, 0, 20, 0, 40, 15]
  # L1=5 физическое (энтропия телесна), L8=15 абстрактный принцип
  mass: 195
  temperature: 3
  state: STATE_LOCKED
```

**Архитектурный нюанс:** T6 — основа для дилемм типа V (аксиогенных, `Dilemmas_V1_0.md`): если событие необратимо и это создаёт конфликт, стандартного разрешения нет. Именно необратимость превращает компромисс в дилемму.

---

### T7 — Horizon (горизонт)

**Философия:** Граница обозримого во времени. За горизонтом прошлого — забытое; за горизонтом будущего — неопределённое. Горизонт — не стена, а предел разрешения: дальше система не "видит" детально. Архетип: "Дальше не знаю".

```yaml
- id: "time_horizon"
  word: "горизонт"
  aliases: ["horizon", "предел памяти", "planning horizon", "видимое будущее", "давно"]
  tags: ["primitive", "time", "t7_horizon"]
  position: [7000, 6000, 13000]
  # Z=13000 высокая Воля: горизонт — это всегда напряжение между тем что знаешь и нет
  shell: [0, 0, 0, 0, 25, 0, 35, 15]
  # L5=25 когнитивный (планирование требует осознания горизонта)
  # L8=15 абстрактный (горизонт — концепция, не восприятие)
  mass: 180
  temperature: 7
  state: STATE_LOCKED
```

**Связь с ActivityTrace:** SHORT (16) / MID (64) / LONG (256) — это три горизонта памяти. T7 — их семантический аналог. Когда LONG-буфер "видит" паттерн, это и есть работа с горизонтом.

## 3. Композиция темпоральных структур

### 3.1 Нарратив = цепочка Before/After

Простейшая темпоральная структура — линейная: A→B→C где каждая пара связана T1/T2. FrameWeaver кристаллизует такую цепочку в Frame "событийная последовательность" (C2-уровень).

### 3.2 Ритм = Duration + Periodic

Когда Duration стабильна и Periodic активна одновременно — FrameWeaver кристаллизует "ритмический паттерн". Примеры: метр в поэзии, пульс в музыке, cadence в речи.

### 3.3 Дилемма необратимости

T6 (Irreversible) + высокая активация двух несовместимых ценностей → диагностируется Дилемма V (аксиогенная). Подробнее: `Dilemmas_V1_0.md`.

### 3.4 Горизонт планирования как Frame

T7 активируется когда ActivityTrace.long обнаруживает паттерн "на больших масштабах". Frame "цель" — это T7 + val_meaningful + конкретный якорь.

## 4. Связь с другими подсистемами

### 4.1 С Writing

Время пронизывает письменность: нарратив, грамматическое время глагола, "вчера сказал что сегодня". TextPerceptor при обработке маркеров времени ("после", "пока", "никогда") активирует Time-примитивы через alias-lookup.

### 4.2 С Mathematics

Пересечение: предел `lim` имеет T4 (длительность бесконечного приближения), производная — мгновенное T4→0, дифференциальные уравнения — T5 (периодические решения). Эти связи кристаллизует FrameWeaver через co-activation.

### 4.3 С Values и Morality

T6 (необратимость) придаёт ценностным конфликтам особую остроту. Ценность нарушена — и это уже нельзя отменить. Эта комбинация — фундамент морального сожаления и аксиогенеза.

### 4.4 С Universal Grounding Stack

Time-примитивы — часть UGS Фазы 1 ("языковые слои"). `discourse_primitives.yaml` (переходы topic_intro → assertion → conclusion) — это T1/T2 на уровне дискурса. `pragmatic_primitives.yaml` — речевые акты имеют темпоральный контекст (обещание = T2, угроза = T2+T6).

## 9. Домены AshtiCore

| Примитив / процесс | Домен | Причина |
|---|---|---|
| T1–T7 (все примитивы) | **D0 SUTRA** | STATE_LOCKED якоря |
| T6 — Irreversible | **D3 CODEX** | закон-ограничение: прошлое нельзя изменить, абсолютное правило |
| T5 — Periodic | **D7 DREAM** | циклы рождаются из фоновой рекомбинации, не задаются явно |
| T3 — Simultaneous | **D10 MAYA** | синтез нескольких потоков в единый момент = проекция |
| T7 — Horizon | **D5 PROBE** | исследование за пределом горизонта — активный зонд |
| ActivityTrace паттерны | **D4 MAP** | подтверждённые темпоральные факты о мире |
| TransitionMatrix (decay) | **D7 DREAM** | фоновая оптимизация статистики переходов |
| Неопознанный темп. паттерн | **D8 VOID** | не вписывается ни в один цикл → датчик аномалии |
| FatigueStore переполнение | **D8 VOID** | когда усталость критична → среда стала непознаваемой |

**Ключевое соответствие:** T6 (Irreversible) = D3 CODEX — оба задают **абсолютные ограничения**, которые не пересматриваются в рантайме. Это объясняет почему необратимость — основа аксиогенных дилемм: нарушить D3 CODEX можно только через аксиогенез.
