## Фаза 1 — Кристалл (Anchor_Crystal_V1_0)

*Аккумулятор. Стартовая палитра графемных якорей по формуле.*

### C1 — charset ru_en_base.yaml

`config/charsets/ru_en_base.yaml`

Кириллица + латиница + цифры 0–9 + базовая пунктуация + математические операторы.  
Для каждой графемы: ранг частоты (из открытых словарей ru/en), класс (гласная/согласная/цифра/пунктуация), число штрихов.  
~150–200 графем. Это входные данные компилятора.

### C2 — axiom-seed: Crystal Layout

Новый крейт `crates/axiom-seed/`:

```
src/
  layout/
    crystal.rs      — формулы: 8 слоёв по d, полярный веер (θ=природа, r=частота), луч цифр
    collision.rs    — проверка дистанции до существующих якорей
  charsets/         — ru_en_base.yaml и будущие
  compiler.rs       — CLI: axiom-seed compile --layout crystal
```

Детерминизм: одинаковый charset + region → одинаковые позиции всегда.

### C3 — FrameComposition: раскрыть C5_Plus

`crates/axiom-experience/` или где живёт enum `FrameComposition`.

`C5_Plus` → `C5` / `C6_Meta` / `C7_Reserve`.  
Согласовать со всеми существующими использованиями.

### C4 — Регион кристалла (с chrnv)

Выбрать `origin` и `size` по карте якорей — свободная зона, не пересекающаяся с осевыми/субсистемными якорями.  
Коллизия-проверка в компиляторе (минимальная дистанция до существующих якорей, порог конфигурируем).  
Ориентир: куб ~8000–10000 на сторону.

### C5 — Компиляция + boot-инъекция

```bash
axiom-seed compile --layout crystal \
    --charset config/charsets/ru_en_base.yaml \
    --region config/crystal_region.yaml \
    → seeds/crystal_c0.yaml
```

Boot-инъекция: anchor-режим (как существующие якоря). Слой C0 на поверхности.

### C6 — OBS-прогон: проверить кристалл

Частые графемы в ядре слоя C0, секторы природы с зазорами (нет коллизий цифра/буква), слой ≠ SutraDepth.  
Первая проверка: графемный слой матчится, co-activation работает.
