# AXIOM — REPAIR-01: координаты ≠ природа

**Дата:** 2026-06-13
**Статус:** ОДНОРАЗОВАЯ инструкция. После выполнения: §1 перенести в INVARIANTS.md, файл УДАЛИТЬ.
**Согласовано:** chrnv + Opus
**Суть:** разделить «ГДЕ» (координаты, чистая близость) и «ЧТО» (природа, философские оси). Сейчас они смешаны — это ошибка уровня «токены скатываются к нулю», отложенная на потом. Чиним сейчас, пока не вросла глубже.

---

## §1 — НОВЫЕ ИНВАРИАНТЫ (после ремонта → в INVARIANTS.md)

```
N1. Координаты [0..32767] положительные. Позиция несёт ТОЛЬКО близость
    (гравитация, грид, резонанс). Позиция НИКОГДА не несёт философию.

N2. Октант/природа токена и Frame вычисляются ТОЛЬКО из:
    - физической подписи (mass / valence / temperature)
    - происхождения (origin_domain)
    НИКОГДА — из знаков или величин координат.

N3. Природу (+/-,+/-,+/-) несут ТОЛЬКО 8 ASHTI-доменов (101–108) —
    как внешнее свойство домена целиком (мембранные профили).
    Природа НЕ распространяется внутрь куба домена (никаких
    «аполлонических углов» внутри — иначе фрактал октантов).
    SUTRA (100) — библиотека, EXPERIENCE (109) — память,
    MAYA (110) — вход: ВНЕ октантной философии.

N4. Центр куба = (16384, 16384, 16384). Точка (0,0,0) — УГОЛ, не центр.
    Глобальных центров притяжения нет; если где-то нужен — это середина
    куба, не ноль.

N5. EXECUTION (101) — основной/главный домен (primary processing).

N6. DREAM-фаза наступает по СОСТОЯНИЮ (простой/idle, в будущем —
    зарядка/расписание суток), НЕ по интервалу тиков. Интервальный
    запуск сна допустим только как тестовый режим OBS (явный флаг).
```
